利用hugo+github pages+stack模板快速架設部落格

應該有蠻多人有聽過hugo或hexo這類靜態網頁產生器用來搭配github pages架設部落格的案例，甚至有不少常看到的個人部落格就是利用這種方式搭建的。接下來我們就利用stack模板來學習如何快速建立這樣的網站吧！

Jan 28, 2024
閱讀時間: 1 分鐘

利用stack模板創建repo

  1. 進入stack模板頁面
  2. 按Use this tmeplate
  3. 選擇 create a new repository

  1. 在 repository name 輸入 使用者名稱.github.io

  1. 按下 create repository

新增codespace及基本設置

  1. 進入剛剛創建的repo
  2. 按下 code ，並選擇 codespace ，按下"+“符號新增 codespace

  1. 等待創建 codespace

  2. 在終端機輸入 hugo server ，測試網站

  3. 修改設定檔config.toml及其他設定檔

新增頁面

  • 在 post 新增 blog 文章
  • 在 page 新增網站頁面

網站上線

  1. 在終端機輸入hugo

  2. 進入原始檔控制頁面

  1. 將更動包成commit

記錄此次commit的更動，並按下確認

  1. 同步到 github

  1. 等github action 跑完

  2. 到 settings 的 pages 頁面修改渲染網頁來源至 gh-page 分支

  3. 再等 github action 跑完就大功告成

結語
